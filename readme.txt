安装说明

  * 将本目录所有文件通过FTP工具上传至服务器
  
  * 将服务器以下目录文件属性设置为777可写模式
      /data/
      /data/authen/
      /data/caches/
      /data/configs/
      /data/uploads/

  * 访问页面 http://服务器域名/admin.php 进入网站管理中心
     超级管理员帐号:admin 默认密码:888888

     前往"站务管理"页面,点击左侧"安装维护"栏目:
        1.点击"文件夹权限",检查运行目录权限配置是否正确.
        2.点击"数据库配置",按服务器接入商所提供的MySQL数据库连接信息进行填写并保存配置.
        3.点击"初始化重置", ( *注意:只在网站初次安装时执行本操作!!! )
           阅读操作须知并同意条款后进行数据库初始化重置操作.
     依次执行以上步骤即可完成网站安装.
